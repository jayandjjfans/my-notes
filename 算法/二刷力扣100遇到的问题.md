# 二刷力扣100遇到的问题

##### 最长不重复字串

更新left位置时，写成了**left=map.get(s.charAt(i))+1;**

实际上**start=Math.max(start,map.get(s.charAt(i))+1);**

忽略了当map中存在某char时，可能left已经跳过了它对应的位置。

```java
            }
        }
```
